# Calculator

## Установка

1. Склонируйте репозиторий:

   ```bash
   git clone
   cd calculator
   ```

   Установите зависимости:

npm install

Запуск

npm start

Приложение будет доступно по адресу http://localhost:3000

Тестирование npm test

Статический анализ
Для запуска ESLint:

npm run lint

Для запуска Stylelint:

npm run stylelint
